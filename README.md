# Network-Intrusion-Detection-using-machine-learning
This project implements a Machine Learning-based Network Intrusion Detection System (NIDS) designed to identify and classify malicious network traffic. Using the CICIDS2017 dataset, I collected, cleaned, and preprocessed network traffic data before training and evaluating classification models such as Naive Bayes and Support Vector Machine (SVM). The models were assessed using performance metrics including accuracy, precision, recall, F1-score, and confusion matrices to determine their effectiveness in detecting cyberattacks.
<br>
I built this project to explore the application of Machine Learning in cybersecurity and gain practical experience with network traffic analysis and threat detection. Through this project, I learned data preprocessing, feature engineering, model evaluation, and the fundamentals of intrusion detection systems. It strengthened my understanding of both Machine Learning and cybersecurity while demonstrating how predictive models can be used to identify and mitigate potential network threats.
<br>
Technologies Used:- Python, Pandas, NumPy, Scikit-learn, Jupyter Notebook
